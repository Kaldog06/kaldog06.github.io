Instructions


1. Make sure your Minecraft Launcher is fully closed

2. Download the Fabric Installer from the link below

3. Run the Fabric Installer

4. Change the version to 1.19.4

5. When this is done, press OK

6. Close the installer


7. Open your Minecraft Launcher

8. Press the "Installations" tab on the top

9. While hovering over the new "fabric-loader-1.19.4" profile, press Play

10. When the game has finished loading completely, close it

11. Go back to the "fabric-loader-1.19.4" under "Installations"

12. Instead of pressing Play, click on the folder icon next to it

13. Open the mods folder

14. Download the "server mods.zip" from below

15. Right click the downloaded file and click "Extract All..."

16. Open the newly generated "server mods" folder

17. Copy EVERYTHING from this folder into the mods folder from step 13
